# Marvel Profile

It is a practice of basic app which makes use of the Marvel API to recreate and create projects based on the best super Heroes.
This project is carried out:

* Swift = v5.0
* Xcode = v12.2
* Design pattern = MVVM
* include basic UnitTest
* include cocoaPods
